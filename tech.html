Technical Background

Time Series Forecasting

Time series forecasting is the use of a model to predict future values based on previously observed values.

Test for Stationarity

  • Stationarity- Constant Mean and Constant Variance in the data.
  • Test - Augmented Dickey Fuller(ADF) Test
  • Results of ADF Test- (i) p-value (ii) Test statistics (iii)Critical values.
  • If p- value is greater 0.05 and Test-statistics is smaller Critical values implies that Series is Stationary.
  • Use differencing till the series becomes stationary.

The ARIMA Model - (The Most Popular Forecasting Technique)

  • ARIMA stands for Autoregressive Integrated Moving Average.
  • It consists 3 main building blocks
    1. Auto Regressive Part
    2. Moving Average Part
    3. Integrated Part

Auto Regressive Model (AR Model)

  • A regression model where current values depend on the past values of itself.
  • Equation: Yt= β1 + ϕ1Yt-1 + ϕ2Yt-2 + … + ϕpYt-p
  • ϕ values- Weights of corresponding lags.
  • β - A constant
  • Partial Autocorrelation is observed in AR Model (Direct Correlation)

Moving Average Model (MA Model)

  • Accounts for analysis of errors in predicting the previous values so that we can make a better estimate for the current observation.
  • Equation: Yt= β2 + ω1εt-1 + ω2εt-2 + … + ωqεt-q + εt
  • ω : Weights of corresponding error lags.
  • ε : Corresponding Error terms.
  • Auto Correlation is observed in MA Model (Both Direct and Indirect correlation )

Integration(I)

  • Binds the AR and MA Model Together.
  • Non-stationary time series needs to be converted to stationary time series data by differencing.
  • Prediction of the difference of time series data from one time point to another instead of predicting original time series data.
  • Order of differencing - (d)
    • Zt= Yt+1 - Yt ...d=1
    Qt = Zt+1 - Zt ...d=2

ARIMA Model (The Final Version)

  • ARIMA(p,d,q)
  • p = AR lags i.e, Number of lags which have significant correlation with the current observation ( PACF plot is used) .
  • q= MA lags i.e, Number of lag observation errors which have significant impact on current observation( ACF plot is used).
  • d= order of differencing.

SARIMA Model

  • SARIMA =Seasonal ARIMA, is an extension of ARIMA that supports univariate time series data with a seasonal component.
  • There are four seasonal elements that must be configured apart from regular parameters:
    1. P: Seasonal autoregressive order.
    2. D: Seasonal difference order.
    3. Q: Seasonal moving average order.
    4. m: The number of time steps for a single seasonal period.
  • Final Form- SARIMA(p,d,q)(P,D,Q)m .
